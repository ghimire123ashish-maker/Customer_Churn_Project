# Clustering Analysis

This folder contains all clustering analysis deliverables for the Customer Churn Analysis Project.

## Included Files

* Optimal Number of Clusters (Elbow Method)
* Trained K-Means Model
* Cluster Visualisations and Labels

## Techniques Used

* K-Means Clustering
* Elbow Method for optimal cluster selection
* Data visualisation using Python libraries

## Tools Used

* Python
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook
